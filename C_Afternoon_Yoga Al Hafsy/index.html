Skill

Beberapa hal yang saya kuasai dalam bidang Web Development :

1

HTML

Biasanya, penggunaan umum HTML adalah untuk menyusun bagian paragraf, heading, maupun link pada halaman web. Tapi, meskipun susunannya seperti coding, perlu diketahui bahwa HTML bukanlah bahasa pemrograman. HTML tidak dianggap sebagai bahasa pemrograman karena tidak bisa memberikan fungsi yang dinamis.

2

PHP

PHP adalah bahasa pemrograman yang umum dipakai dalam pembuatan dan pengembangan suatu web. Sebetulnya, PHP merupakan singkatan dari PHP: Hypertext Prepocessor. Menurut situs tersebut, PHP adalah bahasa pemrograman yang banyak digunakan secara luas dan secara khusus sesuai untuk pengembangan web.

3

Python

Python merupakan bahasa pemrograman komputer yang banyak digunakan dalam membangun situs aplikasi/software, pengembangan perangkat lunak, ilmu data, dan machine learning (ML). Developer menggunakan bahasa Python karena lebih efisien dan mudah dipelajari serta dapat dijalankan di berbagai platform.
